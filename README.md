# TWRP Builder Using Gitpod

Cara penggunaan :
- git clone https://github.com/Massatrio16/mka
- cd mka
- sudo passwd (buat dan konfirmasi password baru untuk su)
- su (Masukkan passwrod baru yang telah dibuat)
- bash build.sh
